## Run Locally

Clone the project

```bash
  git clone https://github.com/dorsamomeni/GuessWho.git
```

Go to the project directory

```bash
  cd GuessWho
```

Install dependencies

```bash
  pnpm install
```

### Environment Variables

To run this project, you will need to create a `.env` file and add the following environment variables to it:

- `EXPO_PUBLIC_CLERK_PUBLISHABLE_KEY`
  - Can be under Configure -> API Keys within the [Clerk Dashboard](https://dashboard.clerk.com/)

Start the server

```bash
  pnpm dev
```

Follow the on-screen instructions and use either Expo Go or iOS Simulator
